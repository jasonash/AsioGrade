## 1. Design Principles

### 1.1 Visual Style

| Aspect | Decision |
|--------|----------|
| Overall Feel | Warm, friendly, approachable |
| Color Mode | Dark (default) and Light modes |
| Typography | Clean, readable, generous sizing |
| Spacing | Comfortable whitespace, not cramped |
| Corners | Rounded (8-12px radius) |
| Icons | Friendly, not overly technical |

### 6.2 Key Navigation Patterns

1. **Sidebar** shows courses (expandable to sections)
2. **Clicking a course** shows course-level content (curriculum)
3. **Clicking a section** shows section-level content (students, grades)
4. **Assessments are created** at course level
5. **Assessments are assigned** at section level
6. **Grades are recorded** per section per assignment
